## 入口一：对话式（意图识别驱动）

```
用户在聊天框输入
        │
        ▼
┌──────────────────┐
│    意图识别       │
│                  │
│  Bot分析用户话： │
│  · 要做什么？    │
│  · 对什么商品？  │
│  · 有什么要求？  │
└────────┬─────────┘
         │
    ┌────┼────┬──────┬───────┐
    ▼    ▼    ▼      ▼       ▼
  生图  生视频 图+视频  查素材  收藏
    │    │    │      │       │
    ▼    ▼    ▼      ▼       ▼
  自动串联对应流水线执行
  中间结果反馈给用户确认
```

**对话示例：**

```
用户：帮我做一张白色运动鞋的白底图
Bot：好的！识别到：
     商品：白色运动鞋
     类型：白底主图
     直接用预置模板生成，稍等...
     → [出图3张]

用户：第二张不错，帮我做成展示视频
Bot：好的，用这张图生成展示视频：
     → [匹配展示型视频模板]
     → [出视频]

用户：收藏一下这个图片模板
Bot：已收藏为"运动鞋白底图模板"，下次同品类会自动推荐。
```

## 两种入口对比

```
┌─────────────────┬─────────────────────┬─────────────────────┐
│                 │    快速生成          │    自定义生成         │
├─────────────────┼─────────────────────┼─────────────────────┤
│ 操作方式        │ 选模板→填信息        │ 逐步操作每个节点     │
│                 │ →一键出结果          │ 每步做完再选下一步   │
├─────────────────┼─────────────────────┼─────────────────────┤
│ 适合场景        │ 日常高频、标准商品   │ 特殊需求、精细控制   │
├─────────────────┼─────────────────────┼─────────────────────┤
│ 用户控制度      │ 低(选模板就完事)     │ 高(每步自己决定)     │
├─────────────────┼─────────────────────┼─────────────────────┤
│ 图片生成        │ 预置模板直接出图     │ 可描述+可选模板      │
├─────────────────┼─────────────────────┼─────────────────────┤
│ 视频生成        │ 选模板自动连贯生成   │ 单独一步，做完图片   │
│                 │                     │ 再决定要不要做视频   │
├─────────────────┼─────────────────────┼─────────────────────┤
│ 图片处理        │ 不含(直接出图)       │ 可选去背景/换背景    │
│                 │                     │ /调尺寸/加水印       │
├─────────────────┼─────────────────────┼─────────────────────┤
│ 耗时            │ 快(1次等待)          │ 慢(多步但更精准)     │
└─────────────────┴─────────────────────┴─────────────────────┘
```

## 核心能力(流水线)汇总

```
A. 商品信息采集 ── 提取商品名/品类/卖点 → 结构化商品卡
B. 图片生成 ────── 预置模板 or 描述 → 生图API → 3-4张候选
C. 视频生成 ────── 预置脚本 or 自定义描述 → 视频API → 5-15s
D. 关键词补全 ──── 知识库检索 → 推荐热词/卖点词
E. 模板收藏 ────── 保存满意Prompt/脚本 → 写入知识库
F. 素材查询 ────── 查历史素材和收藏模板
G. 去背景 ──────── 调用去背景API → 输出透明底图
H. 图片编辑 ────── 换背景/调尺寸/加水印
I. 平台切换 ────── 当前Prompt + 目标平台 → 新Prompt + 新图片
J. 链接提取 ────── 商品URL → 商品名/价格/卖点/主图/来源平台
```

> 图片编辑中的换背景、调尺寸、加水印需要真实图片处理API或生图插件配合，流程图里的 Code 节点只代表参数整理和编排，不代表直接完成图片二次处理。
