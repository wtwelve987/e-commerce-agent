# 六、Bot模型配置

## 6.1 模型选择

在左侧「项目管理」中打开目标Bot/项目，在Bot配置页的「模型」下拉框中选择：

| 推荐模型 | 说明 |
|---------|------|
| 豆包·1.6·lite | 默认选择，用于对话引导、普通回复、工作流编排 |
| DeepSeek-V3.2 | 需要更强推理时切换，用于复杂意图判断或规则分析 |

> 注意：Coze 中选择的 Bot 模型主要用于对话入口、意图识别、知识库调用和工作流编排，不建议把它作为商品图片、商品视频等核心生成能力的唯一来源。正式版本建议通过工作流或插件接入火山方舟 API。

## 6.1.1 模型分工建议

| 能力 | 推荐模型来源 | 说明 |
|------|--------------|------|
| 对话引导 | Coze Bot 模型 | 负责追问、确认需求、解释结果 |
| 意图识别 | Coze Bot 模型 | 判断用户要生图、生视频、查素材、切换平台等 |
| 知识库调用 | Coze Bot 模型 | 按需调用 RecallKnowledge |
| 标题/卖点生成 | 火山方舟文本模型 | 更适合做高质量电商文案 |
| 图片 Prompt 生成 | 火山方舟文本模型 | 根据平台规范和模板生成更稳定的 Prompt |
| 图片生成 | 火山方舟图片生成模型 | 负责实际出图 |
| 图片理解 | 火山方舟视觉理解模型 | 负责分析用户上传图片 |
| 视频生成 | 火山方舟视频生成模型 | 负责实际生成营销视频 |

## 6.2 温度与参数

| 参数 | 建议值 | 说明 |
|------|--------|------|
| Temperature | 0.7 | 平衡创意和准确性 |
| Top P | 0.9 | 默认值 |

## 6.3 记忆设置

| 设置项 | 值 | 说明 |
|--------|-----|------|
| 记忆窗口 | 20轮 | 保留足够对话上下文 |
| 自动总结 | 开启 | 长对话时自动压缩历史 |
