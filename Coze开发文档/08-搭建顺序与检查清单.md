## 8.2 搭建检查清单

### Bot基础

- [ ] Bot名称：电商AI商图视频助手
- [ ] 人设与回复逻辑已完整粘贴
- [ ] 开场白已配置
- [ ] 模型已选择（豆包·1.6·lite）
- [ ] 温度设置为0.7
- [ ] 记忆窗口20轮

### 知识库

- [ ] 商品关键词库（7个文件）
- [ ] 预置模板库（10个文件）
- [ ] 平台规范库（6个文件）
- [ ] 文案模板库（3个文件）
- [ ] 我的素材库（1个模板文件）
- [ ] 5个知识库全部关联到Bot

### 生成能力方案

- [ ] 已确定 Demo 阶段使用 Coze 插件，或正式阶段使用 HTTP 请求接火山方舟
- [ ] 如果使用插件：已添加图片生成、视频生成、去背景、网页读取等插件
- [ ] 如果接火山方舟：已准备 API Key、Endpoint、模型接入点和 HTTP 请求节点参数
- [ ] 已明确哪些节点使用 Coze 内置 LLM，哪些节点调用火山方舟

### 工作流

- [ ] 快速生成（7个节点）
- [ ] 图片生成（6个节点）
- [ ] 视频生成（5个节点）
- [ ] 关键词补全（4个节点）
- [ ] 模板收藏（4个节点）
- [ ] 素材查询（4个节点）
- [ ] 去背景（3个节点）
- [ ] 图片编辑（3个节点）
- [ ] 平台切换（5个节点）
- [ ] 链接提取（6个节点）
- [ ] 10个工作流都已添加到主Bot的「技能→工作流」
- [ ] 主Bot人设中已写明各场景调用哪个工作流

### 快捷指令

- [ ] /快速 已绑定「快速生成」
- [ ] /自定义
- [ ] /平台
- [ ] /关键词 已绑定「关键词补全」
- [ ] /我的素材 已绑定「素材查询」
- [ ] /收藏 已绑定「模板收藏」
- [ ] /去背景 已绑定「去背景」
- [ ] /换背景 已绑定「图片编辑」
- [ ] /换平台 已绑定「平台切换」
- [ ] /提取 已绑定「链接提取」
- [ ] /帮助

### 测试

- [ ] 快速生成流程可跑通（生图+可选视频）
- [ ] 自定义生成流程可跑通（逐步操作）
- [ ] 关键词补全可返回正确推荐
- [ ] 去背景功能正常
- [ ] 链接提取可识别平台并提取信息
- [ ] 平台切换可重新生成适配图片
- [ ] 收藏功能可写入知识库
- [ ] 素材查询可检索到收藏内容
- [ ] 各平台尺寸和风格输出正确
- [ ] 快捷指令均可触发对应工作流
