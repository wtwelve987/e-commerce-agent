# 电商AI商图视频助手 — Coze开发文档

> 本文档为 Coze（扣子）平台上的完整搭建指南，所有操作精确到每一步，开发者可严格依据本文档无歧义地完成搭建。

## 总体架构建议

本项目建议采用「Coze 负责编排，火山方舟负责核心生成」的架构。

Coze 不建议作为唯一的核心生成模型使用。Coze 更适合承担对话入口、知识库召回、工作流编排、用户引导和发布渠道承载；商品图片、商品视频、复杂文案和图片理解等核心生成能力，建议通过工作流或插件接入火山方舟 API。

推荐分工：

| 模块 | 推荐承载方 |
|------|------------|
| 用户对话入口 | Coze Bot |
| 人设、开场白、快捷指令 | Coze Bot |
| 知识库召回 | Coze 知识库 |
| 快速生成/自定义生成流程 | Coze 工作流 |
| 平台规范判断 | Coze 工作流 + 知识库 |
| 标题、卖点、Prompt 生成 | 火山方舟文本模型 |
| 商品图片生成 | 火山方舟图片生成模型 |
| 商品图理解/分析 | 火山方舟视觉理解模型 |
| 商品视频生成 | 火山方舟视频生成模型 |
| 素材收藏/查询 | Coze 工作流 + 知识库/数据库 |

详细说明见：[Coze与火山方舟架构说明](10-Coze与火山方舟架构说明.md)。
