# 七、发布渠道配置

## 7.1 Web网页

1. 在左侧「项目管理」中打开目标Bot/项目
2. 点击页面右上角「发布」
3. 选择「Web网页」渠道
4. 点击「发布」
5. 发布完成后获得分享链接

## 7.2 API

1. 在左侧「API & SDK」中进入API接入页面
2. 选择当前Bot/项目对应的API配置
3. 获取 API Endpoint 和 Token
4. 调用方式：
   - Endpoint：`https://api.coze.cn/v3/chat`
   - Method：POST
   - Header：`Authorization: Bearer {token}`
   - Body：`{"bot_id": "{bot_id}", "user_id": "{user_id}", "stream": false, "auto_save_history": true, "additional_messages": [{"role": "user", "content": "{user_message}", "content_type": "text"}]}`

**安全要求**

- `token` 只能放在服务端、边缘函数或受信代理中，不要写进前端代码
- 前端页面如果直接调用 Coze API，等同于把平台凭证暴露给浏览器
- 如果必须做独立前端，请先补一个轻量后端转发层，再由后端代发请求

## 7.3 微信小程序

1. 在左侧「项目管理」中打开目标Bot/项目
2. 点击页面右上角「发布」
3. 选择「微信小程序」渠道
4. 按提示完成微信小程序授权
5. 点击「发布」
