## 5.2 品类枚举值

| 枚举值 | 说明 |
|--------|------|
| `服饰` | 包含服装、鞋帽、箱包 |
| `美妆` | 包含护肤、彩妆、香水 |
| `食品` | 包含零食、生鲜、饮品 |
| `家居` | 包含家装、家纺、厨具 |
| `数码` | 包含3C、配件、智能设备 |
| `母婴` | 包含童装、玩具、喂养 |
| `其他` | 不在上述品类中 |

| 枚举值 | 说明 |

## 5.5 工作流间变量传递关系

```
链接提取工作流
  └─→ product_name, category, selling_points, source_platform
       ├─→ 图片生成工作流（product_desc, category, platform）
       ├─→ 快速生成工作流（product_name, product_info, platform）
       └─→ 平台切换工作流（product_info, target_platform）

图片生成工作流
  └─→ image_urls, prompt
       ├─→ 去背景工作流（image_url = image_urls[0]）
       ├─→ 图片编辑工作流（image_url = image_urls[0]）
       ├─→ 视频生成工作流（image_url = image_urls[0]）
       ├─→ 模板收藏工作流（prompt, template_name, category, platform, type="图片"）
       └─→ 平台切换工作流（current_prompt = prompt）

视频生成工作流
  └─→ video_url, script_content
       └─→ 模板收藏工作流（prompt = script_content, template_name, category, platform, type="视频"）

关键词补全工作流
  └─→ suggested_keywords
       └─→ 图片生成工作流（keywords = suggested_keywords）
```
