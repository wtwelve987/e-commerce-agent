# 三、插件配置

## 3.1 插件是否必须

插件不是必须使用。

Coze 中实现图片生成、视频生成、去背景、网页抓取等能力有两种方式：

| 方式 | 适用阶段 | 说明 |
|------|----------|------|
| 使用 Coze 插件市场插件 | Demo、快速验证 | 配置简单，适合先跑通流程 |
| 在工作流中使用 HTTP 请求或自定义插件接外部 API | 正式版本 | 可接入火山方舟等模型平台，质量和可控性更高 |

如果采用「Coze 负责编排，火山方舟负责核心生成」的架构，则图片生成、视频生成、视觉理解、标题卖点生成等核心能力建议在工作流中通过 HTTP 请求节点或自定义插件接入火山方舟 API，而不是强依赖 Coze 插件市场。

因此，本章插件配置是可选快捷方案，不是必须步骤。正式版本优先参考 [Coze与火山方舟架构说明](10-Coze与火山方舟架构说明.md)。

## 3.2 Coze插件市场快捷方案

在新版 Coze 左侧点击「集成管理」→「插件/工具」→「+ 添加」，搜索并添加以下插件。添加完成后，再回到「项目管理」中的Bot/项目里引用这些插件：

---

### 插件1：图片生成

| 配置项 | 值 |
|-------|-----|
| 插件名称 | `通义万相`（首选）或 `DALL·E 3` |
| 搜索关键词 | 在插件市场搜索「图片生成」或「通义万相」 |
| 用途 | 生成商品图片 |

**通义万相参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `prompt` | string | 是 | 图片描述（英文Prompt） |
| `negative_prompt` | string | 否 | 反向描述（排除元素） |
| `size` | string | 是 | 尺寸，如 `750*750`、`800*800`、`1080*1080`、`1080*1440` |
| `n` | integer | 否 | 生成数量，默认1，推荐3 |

> **注意**：Coze插件市场中通义万相的 `size` 参数格式为 `宽*高`（用星号连接），如 `750*750`、`1080*1440`。

---

### 插件2：视频生成

| 配置项 | 值 |
|-------|-----|
| 插件名称 | `可灵`（首选）或 `Vidu` 或 `Seedance` |
| 搜索关键词 | 在插件市场搜索「视频生成」或「可灵」 |
| 用途 | 图片转视频 |

**可灵参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `image_url` | string | 是 | 输入图片URL |
| `prompt` | string | 否 | 视频描述 |
| `duration` | string | 否 | 时长：`5`或`10`（秒） |
| `aspect_ratio` | string | 否 | 比例：`1:1`、`9:16`、`3:4`、`16:9` |

---

### 插件3：去背景

| 配置项 | 值 |
|-------|-----|
| 插件名称 | `remove.bg`（首选）或「通义去背景」 |
| 搜索关键词 | 在插件市场搜索「去背景」或「remove bg」 |
| 用途 | 去除图片背景 |

**remove.bg参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `image_url` | string | 是 | 输入图片URL |

---

### 插件4：网页抓取

| 配置项 | 值 |
|-------|-----|
| 插件名称 | `网页读取` 或 `Web Scraper` |
| 搜索关键词 | 在插件市场搜索「网页读取」或「web scraper」 |
| 用途 | 从商品链接提取商品信息 |

**网页读取参数**：

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `url` | string | 是 | 商品页面URL |

---
