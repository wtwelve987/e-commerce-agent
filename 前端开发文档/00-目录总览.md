## 快速开始

1. 先读 `01-项目结构与路由设计` 了解整体架构
2. 按 `12-搭建顺序与检查清单` 中的步骤顺序开发
3. 页面生成使用 `11-Vibe Coding实施指南` 中的Prompt
4. 组件开发参照 `03-组件规格说明` 逐个实现
5. UI和交互规范在 `09-UI设计规范` 和 `10-交互反馈规范` 中查阅

## 技术栈

| 层面 | 技术 |
|------|------|
| 页面生成 | Coze Vibe Coding（自然语言生成） |
| 前端框架 | Coze UI Builder 自定义页面 |
| 后端能力 | Coze Bot + 工作流 + 知识库 + 插件 |
| 认证 | Coze 内置用户系统 |
| 部署 | Coze 托管 |
