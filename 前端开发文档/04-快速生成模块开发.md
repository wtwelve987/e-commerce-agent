# 04 — 快速生成模块开发

## 一、模块概述

快速生成是4步引导式表单，用户依次选择平台→品类+风格→填写商品信息→选择视频模板，最后一键生成。

```
StepProgress: ①平台 → ②品类+风格 → ③商品信息 → ④视频 → 结果
```

---

## 二、4步表单详细设计

### Step 1: 选择平台

**组件**：`PlatformSelector`（mode=`card`）

**数据**：
- 6个平台选项，选中后显示平台规范摘要
- 选中值写入 `AppState.currentPlatform`

**交互**：
- 点击平台卡片 → 高亮+勾选 → 下方显示规范摘要
- 规范摘要包含：主图尺寸、视频比例、风格特点、热词偏好
- 默认选中 `拼多多`

---

### Step 2: 选择品类 + 图片风格

**品类选择**：

| 品类 | 图标 |
|------|------|
| 服饰 | 👗 |
| 美妆 | 💄 |
| 食品 | 🍪 |
| 家居 | 🏠 |
| 数码 | 📱 |
| 母婴 | 🍼 |
| 其他 | 📦 |

**图片风格选择**（使用 `TemplateCard` 组件）：

| 风格 | 说明 | 推荐平台 |
|------|------|---------|
| 白底主图 | 纯白背景，商品居中 | 拼多多、京东 |
| 场景图 | 生活化场景，氛围感 | 抖音、小红书 |
| 模特图 | 人物穿戴展示 | 服饰品类推荐 |
| 促销图 | 促销信息+价格标签 | 拼多多、快手 |
| 详情图 | 细节特写，多角度 | 淘宝、京东 |

**交互**：
- 选中品类后，图片风格根据品类+平台智能推荐（调用知识库）
- 图片风格以 `TemplateCard` 卡片展示，带缩略图预览
- 品类写入 `AppState.productInfo.category`
- 风格写入 `AppState.quickProgress.selectedStyle`

---

### Step 3: 填写商品信息

**表单字段**：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 商品名称 | 文本输入 | 是 | 如"白色运动鞋" |
| 颜色/规格 | 文本输入 | 否 | 如"42码/黑色" |
| 核心卖点 | 文本输入 | 是 | 如"轻便透气、减震回弹" |
| 关键词 | KeywordTags | 否 | 推荐词点击采纳 |

**链接提取（可选）**：

```
── 或 粘贴商品链接 ──
链接: [________________] [提取]
```

- 粘贴URL后点击"提取"
- 调用链接提取工作流
- 返回结果自动填充上方表单字段
- 提取后显示确认信息："从链接提取到：商品名xxx，价格xx，对吗？"

**关键词推荐**：

- 使用 `KeywordTags` 组件
- 根据当前平台+品类，调用关键词补全工作流获取推荐词
- 用户点击采纳或全部采纳

**数据写入**：
- 商品名称 → `AppState.productInfo.name`
- 卖点 → `AppState.productInfo.sellingPoints`
- 关键词 → `AppState.productInfo.keywords`

---

### Step 4: 选择视频模板（可选）

**选项**（使用 `TemplateCard` 组件）：

| 模板 | 说明 | 推荐平台 |
|------|------|---------|
| 展示型 | 360°旋转展示，节奏适中 | 通用 |
| 对比型 | 使用前后对比，突出效果 | 美妆、家居 |
| 开箱型 | 拆箱体验，真实感 | 数码、母婴 |
| 促销型 | 价格突出，限时秒杀 | 拼多多、快手 |
| 暂不生成视频 | — | — |

**交互**：
- 默认选中"暂不生成视频"
- 选中模板后写入 `AppState.quickProgress.selectedVideoTemplate`

---

### 一键生成

**按钮文案**：`🚀 一键生成`

**调用逻辑**：

```javascript
async function handleQuickGenerate() {
  // 1. 组装参数
  const params = {
    user_id: coze.getUserId(),
    platform: state.currentPlatform,
    category: state.productInfo.category,
    image_style: state.quickProgress.selectedStyle,
    product_name: state.productInfo.name,
    product_info: state.productInfo.sellingPoints.join('、'),
    keywords: state.productInfo.keywords.join(','),
    video_style: state.quickProgress.selectedVideoTemplate,
  };

  // 2. 显示骨架屏
  showSkeleton();

  // 3. 调用工作流
  try {
    const result = await coze.runWorkflow({
      workflowId: QUICK_GEN_WORKFLOW_ID,
      parameters: params,
    });

    // 4. 写入结果
    state.currentResult = {
      images: result.images,
      video: result.video || null,
      selectedImageIndex: 0,
      prompt: result.prompt,
    };

    // 5. 切换到结果视图
    showResultPreview();
    showToast('success', '生成成功！3张图片可选');

  } catch (error) {
    showToast('error', '生成失败，请重试');
  } finally {
    hideSkeleton();
  }
}
```

## 三、结果预览

生成完成后，StepProgress变为"完成"状态，主内容区展示 `ResultPreview` 组件。

### 结果页操作流程

```
3张图片展示 → 用户点击选中一张
    │
    ├── [下载原图] → 触发浏览器下载
    ├── [下载视频] → 触发浏览器下载（有视频时）
    ├── [去背景]  → 调用去背景工作流 → 更新processedImageUrl
    ├── [换背景]  → 弹出颜色选择器 → 调用图片编辑工作流
    ├── [调尺寸]  → 弹出平台尺寸列表 → 调用图片编辑工作流
    ├── [加水印]  → 输入水印文字 → 调用图片编辑工作流
    ├── [收藏模板] → 输入模板名 → 调用模板收藏工作流
    ├── [生成视频] → 跳转到视频模板选择 → 调用视频生成工作流
    ├── [切换平台] → 弹出平台切换确认 → 调用平台切换工作流
    └── [再来一次] → 重置表单，回到Step1
```

## 四、状态管理

### 快速生成相关状态

```javascript
// 进入快速生成时初始化
function initQuickGen() {
  state.quickProgress = {
    currentStep: 1,
    selectedStyle: '',
    selectedTemplate: '',
    selectedVideoTemplate: null,
  };
  state.productInfo = {
    name: '',
    category: '',
    sellingPoints: [],
    keywords: [],
  };
}

// 步骤前进
function nextStep() {
  if (state.quickProgress.currentStep < 4) {
    state.quickProgress.currentStep++;
  }
}

// 步骤后退
function prevStep() {
  if (state.quickProgress.currentStep > 1) {
    state.quickProgress.currentStep--;
  }
}

// 表单校验（每步离开时）
function validateStep(step) {
  switch (step) {
    case 1: return !!state.currentPlatform;
    case 2: return !!state.productInfo.category && !!state.quickProgress.selectedStyle;
    case 3: return !!state.productInfo.name;
    case 4: return true; // 视频可选
  }
}
```
