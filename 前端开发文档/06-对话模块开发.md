# 06 — 对话模块开发

## 一、模块概述

对话视图嵌入 Coze Bot 对话组件，用户自由输入，Bot 进行意图识别后自动串联工作流。

## 三、Coze Bot 对话组件嵌入

### 嵌入方式

在 Coze 自定义页面中，使用 Coze SDK 的对话组件：

```javascript
// 初始化Bot对话
const chat = coze.createChat({
  botId: 'YOUR_BOT_ID',
  userId: coze.getUserId(),
  container: document.getElementById('chat-container'),
});

// 监听Bot回复
chat.onMessage((message) => {
  if (message.type === 'workflow_result') {
    // Bot触发了工作流，展示结果
    handleWorkflowResult(message.data);
  }
});
```

### 对话中的结构化数据

Bot回复中可包含结构化数据（图片URL、视频URL等），前端解析后以卡片形式展示：

| Bot回复类型 | 前端渲染 |
|------------|---------|
| 纯文本 | 聊天气泡 |
| 图片列表 | ResultPreview 卡片 |
| 关键词推荐 | KeywordTags 标签 |
| 平台选择 | PlatformSelector 内联 |
| 确认选项 | 按钮组 |

## 四、快捷指令

在输入框上方显示快捷按钮：

| 指令 | 文案 | 效果 |
|------|------|------|
| `/快速` | ⚡快速生成 | 进入快速生成流程 |
| `/自定义` | ✏️自定义生成 | 进入自定义生成流程 |
| `/关键词` | 💡关键词 | 查询当前品类关键词 |
| `/我的素材` | 📁素材库 | 跳转素材库 |
| `/帮助` | ❓帮助 | 显示使用说明 |

## 五、意图识别后的工作流串联

Bot自动识别用户意图后，前端无需手动调用工作流，Bot内部自动串联。但前端需要处理Bot返回的结果：

```javascript
// Bot工作流结果处理
function handleWorkflowResult(data) {
  switch (data.type) {
    case 'image_result':
      // 展示图片结果
      renderImageResult(data.images, data.prompt);
      break;
    case 'video_result':
      // 展示视频结果
      renderVideoResult(data.video_url, data.script);
      break;
    case 'keyword_result':
      // 展示关键词推荐
      renderKeywordTags(data.keywords);
      break;
    case 'product_info':
      // 展示提取的商品信息
      renderProductInfo(data);
      break;
  }
}
```

## 六、对话与操作面板的联动

对话中的操作可跳转到对应模块：

| 对话中的操作 | 跳转 |
|-------------|------|
| 用户说"快速生成" | → 跳转 `/quick` |
| 用户说"自定义" | → 跳转 `/custom` |
| 用户说"查素材" | → 跳转 `/material` |
| Bot返回图片结果后用户选图 | → 在对话内展示 ResultPreview |
| 用户说"去背景" | → 在对话内触发去背景工作流 |
| 用户说"换平台" | → 在对话内触发平台切换工作流 |

对话模式的核心优势：所有操作在对话内完成，无需跳转页面。
